# Math Puzzle Game

This is a simple command-line Python game where players solve random math problems (addition, subtraction, and multiplication) within a time limit. After answering 10 questions, the game shows the player's score and offers an option to play again.

## Features:
- Random addition, subtraction, and multiplication questions
- Time limit for each question (10 seconds)
- Score tracking after 10 questions
- Option to play again after each round

## How to Run:
1. Clone the repository.
2. Ensure you have Python installed.
3. Run the script by navigating to the folder and typing `python math_game.py` in the terminal.

Enjoy the game!
